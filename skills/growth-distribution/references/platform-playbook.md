## 小红书

### CES 评分公式 (Community Engagement Score)
```
CES = 点赞×1 + 收藏×1 + 评论×4 + 转发×4 + 关注×8
```
- ≥8字含场景描述的高价值评论 = 普通评论权重×3
- 秒赞率>50% → 系统判定刷量 → CES扣1.5分
- 收藏多但无真实评论 → 疑似刷量

### 流量池晋级 (2026 星云5.0)
| 阶段 | 曝光 | 晋级条件 |
|------|------|---------|
| 初始池 | 200-500 | 每5分钟实时赛马 |
| 图文晋级 | — | 点击率≥8%, 互动率≥5% |
| 视频晋级 | — | 完播率≥45% |
| 搜索流量 | 占30-65% | 2026年权重提升40%（长尾效应） |
| 推荐流量 | 占~60% | 发现页算法推荐 |

### 内容策略
- 前3秒跳出>45% = 直接限流
- 图文滑动未超3屏 = "无效阅读"
- 搜索SEO关键词: 亲密协议/口头承诺/恋爱安全感/数字存证
- 标签全中文（英文标签零搜索量）
- 标题≤20字

### DYA 小红书执行
- 帖尾加提问引≥8字评论（CES×3加成）
- 发布后1小时密集互动
- 每条评论都详细回复（也算评论数）
- 视频>图文（推荐权重更高）
- Day4开发者故事角度已验证有效（8赞8评251浏览）

## 抖音

### 三级火箭模型 (2025重构)
1. **创作者分级**: S/A/B/C级账号获不同流量加持
2. **流量池晋级**: 冷启动500-1000曝光 → 完播率≥40% → 下一级
3. **动态校准**: 每日数据调整推荐策略

### 算法特征
- 24h爆量逻辑 → 7-15天慢热成长（重大变化）
- "行为预测"取代"打标签"（神经网络计算互动可能性）
- 三级标签: 核心+辅助+潜力
- 互动质量评分: 评论点赞比、购物转化率影响推荐
- 完播率是"入场券"非"登顶证"（后续看互动深度）

### DYA 抖音执行
- 60s广告片适配9:16
- 垂直定位: 恋爱/亲密关系/独立开发
- Dou+投放¥50-100测试（智能推荐→恋爱/情感）
- 标题用话题引争议但不标题党

## TikTok

### Follower-First Testing (2026新机制)
1. 视频先推给粉丝
2. 粉丝互动好 → 才推非粉丝
3. Interest Graph > Social Graph（兴趣匹配>社交关系）

### 权重排序
分享/收藏 >> 评论 >> 点赞

### 关键数据
| 指标 | 值 |
|------|-----|
| 初始测试池 | 200-500人 |
| 病毒阈值 | 72h内100万+播放 |
| 垂直领域爆 | 10-25万也算 |
| 最佳时长 | 60-180秒（2026趋势：长视频更被推荐） |
| 发布频率 | 3-5次/周 |

### DYA TikTok 执行
- 60s广告正好在2026甜蜜区
- Caption嵌入SEO: consent agreement, intimate boundaries, dating safety
- Hook=前3秒: 凝视+「worst morning of my life」
- 原声/原创音频有优势（我们的旁白就是原创）
- 24h间隔发布

## Instagram Reels

### 权重排序
分享(DM转发) >> 收藏 >> 评论 >> 点赞

### 关键规则
- 内容表现 > 粉丝数（200粉好Reel可打败5万粉平庸Reel）
- 完播+重播 = 最强信号
- 原创优先（2026）：其他平台水印被降权
- 7-90秒最佳（60s完美适配）

### DYA IG 执行
- Reels为主（等视频成片后）
- 不带TikTok水印（分别导出）
- Loop设计: D段Logo → 尝试接A段开头形成循环
- Trending audio可考虑（缩短版配热门音频）
- Carousel轮播帖做辅助（截图+品牌故事）
- Trial Reels: 仅推给非粉丝的测试功能

## X (Twitter)

### 权重乘数表

| 行为 | 权重倍数 |
|------|---------|
| 转推 Repost | **20x** |
| 回复 Reply | **13.5x** |
| 主页点击 | 12x |
| 链接点击 | 11x |
| 书签 Bookmark | **10x** |
| 点赞 Like | 1x |

### 关键规则
- 外链降权50-90%（先发内容，链接放self-reply）
- 时间衰减: 每6小时影响力减半
- X Premium: 2-4x曝光加成（$8/月）
- 原生视频 > 外链视频
- 小号有主动扶持政策

### DYA X 执行
- 文字帖+视频分开发（先文字Thread次日视频）
- 原生视频上传（不用链接到YouTube）
- 不放外链（链接放self-reply）
- 争议性观点引RT和Reply
- 考虑X Premium（$8/月性价比极高）
- 黄金时间: 美东9-11am / 欧洲14-16 UTC

## Reddit

### 算法特征
- 前60分钟是关键: 5-7个upvote → 算法放大
- 9:1规则: 每1条推广需9条真实贡献
- AI增强: 高互动帖出现在Google搜索和AI摘要
- 时间衰减: 帖子热度自然下降
- Karma门槛: ~25 karma才能在多数sub发帖

### DYA Reddit 执行（解封后）
- 先攒karma: 真实互动5-7天
- 帖子不放链接，Bio放doyouagree.app
- Builder身份 > CEO身份
- 目标顺序: r/SideProject → r/Futurology → r/UploadTV
- 引发讨论式标题

## 微博

## B站

### 算法特征
- "三连" = B站货币: 点赞+投币+收藏
- 弹幕+评论是独特互动信号
- 完播率对长视频极重要
- 2025更新: 重视原创、互动深度、社交化

### DYA B站执行
- 60s广告适合（用户习惯看完整视频）
- 标题高信息密度: 「一个人做了恋爱协议App」
- 鼓励三连
- 弹幕引导: 开头植入「你会用吗？打1或2」

## 微信视频号

### 算法特征
- 社交+算法双轮驱动
- 朋友点赞 = 强力推荐（出现在朋友推荐中）
- 视频推荐权重 = 图文的3倍
- 更倾向推给非粉丝（鼓励获取新粉）

### DYA 视频号执行
- 社交裂变: 发视频→微信群+朋友圈同步
- 朋友圈预热→1小时后发视频号
- 引导"在看"和转发（视频号最重要按钮）
- 已发Pitch Deck ✅，广告成片后再发完整版

## 视频制作规范（跨平台）

| 要素 | 要求 | 原因 |
|------|------|------|
| 前3秒 | 必须震撼 | 所有平台前3秒跳出>45%=限流 |
| 无水印 | 分别为每平台导出 | IG降权TikTok水印 |
| 双版本 | 无URL版+有URL版 | TikTok/IG无URL, YouTube/X/微博有URL |
| Loop | 结尾尝试接开头 | 循环播放=完播率翻倍 |
| 字幕 | 必须有 | 大量用户静音看视频+算法理解内容 |
| 竖版优先 | 9:16 | 短视频平台标准 |
| 横版备用 | 16:9 | YouTube/B站/X |
